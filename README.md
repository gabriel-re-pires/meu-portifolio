# Portfólio — Gabriel R. Pires

Portfólio pessoal de Engenheiro da Computação e Desenvolvedor Fullstack.
Visual em pixel art / retrô, com troca de tema (laranja, vermelho, azul e rosa)
e modos claro/escuro.

## Tecnologias

- React + TypeScript
- Vite
- Tailwind CSS

## Como rodar

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
```

## Deploy (GitHub Pages)

```bash
npm run deploy
```

## Estrutura

```
src/
  assets/        imagens, certificados e o personagem (gif/avatar por tema)
  components/    Navbar, Hero, About, Projects, Education, Contact, Footer, ...
  contexts/      ThemeContext (troca de cores do site)
  data/          dados dos projetos (software / hardware)
  pages/         Index e páginas internas de projetos
  index.css      design system (tokens, temas e estilos)
```
